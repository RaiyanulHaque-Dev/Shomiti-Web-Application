# Shomiti Management Web Application

A modern web application for managing community **Shomiti (association) operations** including member management, financial transparency, notices, and administrative control.

## Features

* Member registration and management
* Contribution and payment tracking
* Receipt generation for payments
* Notice and announcement system
* Admin dashboard for full control
* Financial transparency for members
* Secure login for administrators

## Technology Stack

* Frontend: React + Vite
* Backend: Node.js + Express
* Database: SQLite
* Deployment: Render / VPS Hosting

## Installation

Clone the repository:

```
git clone https://github.com/RaiyanulHaque-Dev/Shomiti-Management-Web-Application.git
```

Navigate to the project folder:

```
cd Shomiti-Management-Web-Application
```

Install dependencies:

```
npm install
```

Build the project:

```
npm run build
```

Start the server:

```
node server.ts
```

The application will run on:

```
http://localhost:3000
```

## Deployment

This project can be deployed on platforms that support Node.js such as:

* Render
* VPS hosting providers
* Cloud hosting platforms

## Purpose

This system is designed to simplify the management of local associations or committees by digitizing member records, payments, and administrative tasks.
